About StudentSphere

Your complete student management solution for the digital age.

📌 Project Overview

StudentSphere is a full-stack web application developed as the final semester project for Web Technologies (CSC-251) at PMAS-Arid Agriculture University. It demonstrates complete CRUD operations, responsive design, RESTful API, and cloud deployment.

⚙️ Technologies Used

Node.js

JavaScript Runtime

Express.js

Web Framework

MongoDB Atlas

Cloud Database

Bootstrap 5

Responsive UI

Development Timeline

Week 1-2

Learning Node.js & Express fundamentals

Week 3

Database design with MongoDB & Mongoose

Week 4

Building REST API & CRUD operations

Week 5

Frontend with Bootstrap & API integration

Week 6

Testing, debugging & deployment on Render

Key Features

  • Complete Student CRUD (Add, View, Edit, Delete)
  • Responsive mobile-friendly design
  • RESTful API with proper HTTP methods
  • Cloud database (MongoDB Atlas)
  • Live deployment on Render.com
  • Animated counters & modern UI
